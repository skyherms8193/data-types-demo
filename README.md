# Lemmy Data Types 🐶

A lighthearted Python project demonstrating core data types using my dog, Lemmy.

## Overview

This script showcases four fundamental Python data types:

- **List** → Lemmy's daily activities  
- **Tuple** → Lemmy's basic information  
- **Dictionary** → Frequency of daily routines  
- **String** → A sentence about Lemmy with a word count  

## Why this project?

Instead of using generic examples, this project applies programming concepts to something personal and fun.

Because learning is better when it's not boring.

## Concepts Practiced

- Iterating through lists
- Accessing tuple elements by index
- Looping through dictionary key-value pairs
- String manipulation using `.split()` and `len()`

python lemmy_data_types.py
